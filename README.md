## What this project does

This project creates a fully functional proxy site that mirrors the XDrive proxy setup. After deployment, the system becomes your own personal copy, running under your Netlify domain or any custom domain you choose.

Once deployed:

* A brand new clone site is created under your Netlify account.
* You can share your clone with friends, family, or your community.
* You are free to promote and use it as your own project.
* There are no licenses, restrictions, or copyrights from XDrive.
* Everything is open source and safe to use or modify.

This repository is designed so anyone can deploy and run the proxy without any configuration or technical knowledge.

## How to deploy manually

If you prefer not to use the one-click button:

1. Fork or clone this repository
2. Push it to your own GitHub
3. Connect the repository to Netlify
4. Deploy
5. Netlify will automatically detect the Edge Function and activate the proxy
